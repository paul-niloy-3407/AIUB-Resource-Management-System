# AIUB Resource Management System

This is a C# Windows Forms desktop application developed for managing university resources efficiently.

The system allows Faculty, Students, and Clubs to register and book resources while the Admin controls the entire system.

## Features

- User registration and login system
- Admin controlled resource management
- Role-based access for Faculty, Student and Club
- Resource booking system
- Automatic booking list update after confirmation

## Resources

Student Resources:
